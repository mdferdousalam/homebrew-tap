# Homebrew Tap — Markdown Viewer

Homebrew tap for [**Markdown Viewer**](https://github.com/mdferdousalam/md-viewer),
a simple, fast cross-platform Markdown viewer and editor.

## Install

```sh
brew install --cask mdferdousalam/tap/md-viewer
```

## Update

```sh
brew upgrade --cask md-viewer
```

## Uninstall

```sh
brew uninstall --cask md-viewer
```

The cask (`Casks/md-viewer.rb`) is regenerated automatically on each release of the
main project. The app is not code-signed, so the cask clears the macOS quarantine
flag on install for a clean first launch.
